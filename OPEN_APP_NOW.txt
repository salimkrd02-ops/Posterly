## ✨ Troubleshooting

**"RUN.bat not found?"**
- Check HOW_TO_RUN.md for alternative method

**"npm not found?"**
- Install Node.js from https://nodejs.org/ (v18+)

**"Port 3000 in use?"**
- Edit RUN.bat, change last line to:
- `npm run dev -- -p 3001`
- Then visit http://localhost:3001
