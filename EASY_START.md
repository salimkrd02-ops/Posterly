## 🛠️ Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| npm not found | Install Node.js v18+ |
| Port 3000 in use | Use `-p 3001`: `npm run dev -- -p 3001` |
| Module error | Run: `npm install && npm run prisma:generate` |
| File not found | Check path - see EXACT_STEPS.md |
| Slow on first run | npm install takes 2-5 min - be patient |
