## 🎯 Project Overview

**PosterGen:** A full-stack web application for creating and managing professional event posters.

**Core Features:**
- User authentication (signup, login, password reset)
- Event management (create, edit, organize)
- Visual template editor (drag-and-drop)
- Poster generation (dynamic data mapping)
- Multi-format export (PNG, JPG, PDF)
- Admin dashboard (metrics, user management)
- Role-based access (5 roles)

**Technologies:**
- Next.js + React + TypeScript
- PostgreSQL + Prisma
- Tailwind CSS + Fabric.js
- Supabase Storage
- Vercel ready
